Visual Studio Live Share

Desenvolvimento Colaborativo em Tempo Real

Bem-vindo ao Visual Studio Live Share.

Inicie sua equipe em edição e depuração colaborativa com apenas algumas etapas simples:

Por que entrar?

Ao colaborar, é importante que outras pessoas possam identificar quem você é, para ficarem confiantes em compartilhar o acesso ao ambiente de desenvolvimento e código. Depois de entrar, você terá direitos de leitura-gravação de modo que possa editar o código junto com o host.

Para que seja mais fácil começar, você pode entrar usando uma conta Microsoft ou GitHub existente.

Para que seja mais fácil começar, você pode entrar usando uma conta Microsoft ou GitHub existente.

Compartilhar

Clique no botão Live Share no canto superior direito, que compartilhará o seu projeto, e copie uma URL de sessão exclusiva para sua área de transferência.

Você precisará entrar na primeira vez que compartilhar (usando uma conta do GitHub ou Microsoft), o que permite que outras pessoas identifiquem você ao colaborar.

No Windows, talvez lhe peçam que permita que o Live Share abra uma porta de firewall para habilitar conexões ponto a ponto.

Editar

Agora que você está em uma sessão de colaboração, você pode mover o cursor, editar e destacar o texto. Você pode ver outros participantes fazendo o mesmo.

Você pode acessar toda a funcionalidade do Live Share clicando no menu suspenso "Ingressado" ou nos ícones do participante no canto superior direito.

Daqui, você pode ver os participantes na sessão, bem como os servidores e os terminais que foram compartilhados.

  • Mover para outros arquivos no código. Você é capaz de navegar pelo projeto.

  • Concentre-se em um participante e chame a atenção para uma parte do código para a qual você está olhando.

  • Siga qualquer participante em uma sessão clicando no nome dele na parte superior direita.

Depurar

Resolva um problema/pergunta/tarefa com sua equipe em tempo real usando os diversos recursos de depuração colaborativos.

  •  Depure em conjunto, quando em uma sessão de depuração; defina pontos de interrupção e execute etapas em conjunto em tempo real.

  • ☁️ Exiba servidores compartilhados clicando no menu suspenso "Ingressado" para exibir os aplicativos Web, os bancos de dados, etc. do host que estão sendo executados localmente.

  •  Mostre terminais compartilhados clicando no menu suspenso "Ingressado" para exibir ou executar comandos no terminal do host.

Convidar

Envie a URL da sessão aos desenvolvedores com os quais deseja colaborar (até cinco) usando email, Skype, Slack, etc. Veja as informações de segurança para saber mais sobre compartilhamento seguro.

Deseja avaliar o VS Live Share, mas não tem com quem colaborar? Sem problemas. Na verdade, você pode compartilhar com si mesmo de outra instância do VS. Clique aqui para experimentar.

Colaborar

É isso. Quando um "participante" ingressar, você receberá uma notificação e verá o cursor dele aparecer no arquivo aberto no momento. Todos os participantes começarão seguindo você, o que significa que, ao rolar a página ou navegar entre os arquivos, eles acompanharão essas ações. Isso torna mais fácil orientá-los em relação ao problema/pergunta/tarefa no qual vocês começarão a colaborar.

Além disso, para aproveitar ao máximo o Live Share, experimente o seguinte:

  • Mova o cursor, edite e realce texto e veja os participantes fazerem o mesmo
  • Siga um participantee faça com que ele role a página e/ou navegue para outros arquivos
  • Compartilhe uma sessão de depuração pressionando F5 e, em seguida, entrando juntos em tempo real
  • Compartilhe um servidor para que os participantes possam ver um aplicativo Web ou um banco de dados em execução local, entre outros.
  • Compartilhe um terminal e execute comandos de maneira colaborativa

Se você tiver problemas, consulte nossa documentação de solução de problemas para obter possíveis soluções alternativas. Além disso, se você tiver perguntas e/ou comentários, entre em contato com a equipe usando as opções Enviar Comentários no menu Ajuda.
